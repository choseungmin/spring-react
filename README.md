# Spring X React Collaboration

Spring framework 와 React library 를 이용한 게시판 프로그램입니다.

스프링과 리액트의 조합을 코드로 보여주기 위한 예제 프로그램입니다.

**개발 계획**

1. 게시판 개발
2. 캐시
3. 권한관리
4. 게시물 알림 (소켓)
5. 최종 리팩토링

**사용된 언어**

- java 7
- js (es6 +)
- html & css

**사용될 오픈소스**

- spring framework
- spring security
- spring cache
- spring socket
- spring data jpa
- H2 DBMS
- react
- react-redux
- react-router
- Bootstrap (html&css 컴포넌트 프레임워크)
